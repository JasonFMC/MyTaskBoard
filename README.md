# PriorityTasks

A clean, client-side task management app that helps you focus on what matters most. Tasks are automatically sorted by priority (High → Medium → Low), so you always know what to do first.

## ✨ Features

- Add tasks with priority levels (High / Medium / Low)
- Auto-sort: high priority tasks appear at the top
- Mark tasks as complete / undo
- Delete tasks
- Data persists in browser (localStorage) – no backend needed
- Real-time task statistics (total, completed, pending)
- Responsive design, works on mobile and desktop

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, custom properties)
- Vanilla JavaScript (ES6)
- localStorage for data persistence
